About Me

Hi, I'm Noah Sweilem. I'm a 21-year-old Arizona native turned Virginia Tech student. Along my journey, I've spent countless hours hacking away at all kinds of computer projects. I enjoy learning new things, and I work hard to make sure I understand everything I learn.

Portfolio

  • SCAsm

    A React SPA that provides an IDE, emulator, and debugger for assembly programs targeting the single-cycle Simple Computer RISC ISA presented in chapter 8 of Logic and Computer Design Fundamentals (5 ed.) by M. Morris Mano, Charles R. Kime, and Tom Martin.

    • SCAsm
    • Source on GitHub
  • JSFormer

    A React SPA that acts as a replacement for the Image Reformer utility for TI's MSP432 GrLib that provides a graphical interface for downscaling and converting images to C source files. I designed this utility for my own use, but it is now the tool used in the Embedded Systems course (ECE 2564) at Virginia Tech.

    • JSFormer
    • Source on GitHub
  • Kojimaize

    A joke Discord bot designed around a fake Tweet from a Japanese video game designer that quickly became a study into NLP, phonetics, and inter-lingual transliteration.

    • Kojimaize
    • Source on GitHub
